Bouton d'action principal — pilule, libellé en sentence case, verbe d'action (« Enregistrer mon hypothèque »).

```jsx
<Button variant="primary" size="lg">Obtenir mon certificat</Button>
<Button variant="secondary">Voir les offres</Button>
```

Variants : `primary` (vert-600), `secondary` (contour), `ghost` (texte vert), `danger`. Tailles sm/md/lg. Un seul `primary` par vue.
