# UI kit — App de suivi

Tableau de bord client : cartes taux / montant / échéance (mono), carte ambre « fenêtre de renégociation » avec timeline signature, onglets (Mon hypothèque, Offres, Documents, Réglages), liste d'offres avec commission affichée, toast + dialog.

Interactif : changer d'onglet, choisir une offre (dialog), toggles.
